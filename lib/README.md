# Shared Utilities (`/lib`)

This folder contains shared utilities and configuration:

- `constants.ts`: eviction policy enums and defaults.
- `memory.ts`: conceptual KV memory sizing helpers.
- `tokenRendering.ts`: visual token spacing rules.
- `kvModel.ts`: baseline KV data types used by legacy demos.
- `simulator.ts`: deterministic KV grid simulation used by the basic token Ã— layer demo.

Core step logic for the main visualizer lives in `/core`.
